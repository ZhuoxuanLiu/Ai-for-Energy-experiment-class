Python安装

Python简介

Python 是一种高级的、面向对象的编程语言，它的设计注重代码的可读性和简洁性。 它被广泛应用于数据科学、机器学习、人工智能、Web 开发、网络编程等领域，并且在近年来迅速发展和流行。

  • 简洁易读：Python 的语法简单易懂，具有优雅的表达力，使得程序员可以更加专注于问题本身，而不是语言的细节。
  • 大量的开源库：Python 有非常丰富的第三方库和工具，这些库能够完成各种各样的任务，例如：NumPy、Pandas、Scikit-Learn、 TensorFlow等等，可以用于数据分析、科学计算、机器学习、深度学习等领域，能够提高开发的效率。
  • 跨平台：Python 可以运行在多个平台上，包括 Windows、Linux、MacOS 等，这意味着您可以在不同的操作系统上开发和部署应用程序。
  • 社区支持：Python 有一个庞大的开发者社区，这些人共享着他们的经验和代码，帮助解决问题并提供支持。

Miniconda

Miniconda 是 conda 的免费最小安装程序。 它是 Anaconda 的小型引导程序版本，仅包含 conda、Python、它们所依赖的包， 以及少量其他有用的包，包括 pip、zlib 和其他一些包。 使用 conda install 命令从 Anaconda 存储库安装 720 多个额外的 conda 包。 勾选添加到环境变量！ link

新建conda虚拟环境 


										conda create -n ENV_NAME python=3.*.*

激活环境 


										conda activate ENV_NAME

显示该环境下已安装包 


										conda list

退出环境 


										conda deactivate

PIP

pip 是 Python 的一个包管理工具，它可以用来搜索、安装、升级和卸载 Python 包。它是 Python 标准库之外的第三方包， 可以通过安装 Python 或在操作系统的包管理系统中安装来获得。使用 pip，您可以轻松地从 Python 包索引(PyPI) 安装最新版本的包， 并且可以自动处理包之间的依赖关系。除此之外，您还可以使用 pip 安装本地的包，从其他索引中安装包，升级和卸载已安装的包， 以及查看已安装的包列表等等。 link

从 PyPI 安装包 


										pip install <package>

安装指定版本的包 


										pip install <package>==<version>

从requirements.txt文件中安装依赖项 


										pip install -r requirements.txt

卸载指定的包 


										pip uninstall <package>

Conda换源

conda 国内下载第三方库速度较慢，可考虑切换国内源，如清华源、中科大源、阿里云源等。

windows下添加清华源 


										conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
										conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge 
										conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/msys2/
										conda config --set show_channel_urls yes

linux下添加清华源 


										vim ~/.condarc
									

										channels:
										- https://mirrors.ustc.edu.cn/anaconda/pkgs/main/
										- https://mirrors.ustc.edu.cn/anaconda/cloud/conda-forge/
										- https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
										- defaults
										show_channel_urls: true

换回默认源 


										conda config --remove-key channels

Python入门

运行你的第一个python程序

新建文件夹，使用vscode打开，新建文件hello.py，输入以下代码： 


							print("hello world")

运行方式：

  • vscode 右上角运行按钮，执行当前文件
  • vscode 菜单栏，选择运行，选择调试或者非调试运行，选择python
  • vscode 打开终端，在当前目录输入python hello.py

两个注意事项：

  • 使用vscode运行与调试可以创建launch.json文件，可以在里面配置运行环境，比如环境变量、传入参数等
  • 请在右下角选择正确的python环境，否则可能会出现找不到模块的错误

配置文件例子： 


							{
								"version": "0.2.0",
								"configurations": [
									{
										"name": "Python: 当前文件",
										"type": "python",
										"request": "launch",
										"program": "${file}",
										"console": "integratedTerminal",
										"justMyCode": true,
										"env": {},
										"args": []
									}
								]
							}

program选项可替换为其他文件：

  • ${file}代表当前选中文件
  • ${workspaceFolder}代表当前工作区文件夹

args选项可传入参数（使用argparse库解析），env为键值对，可配置环境变量，通过os库获取

Jupyter Notebook 介绍

Jupyter Notebook 是一个基于 Web 的交互式计算环境，支持多种编程语言（包括 Python、R、Julia 等）， 提供了一种方便的方式来创建和共享文档，其中可以包含代码、方程式、可视化图表和文本等多种元素。 使用 Jupyter Notebook，您可以将代码、文档和可视化图表等元素融合在一起， 方便快捷地进行数据分析和交流，同时可以记录和共享整个分析过程。 link

安装结束后直接在终端输入jupyter notebook即可启动：

或使用vscode 安装jupyter扩展后直接新建jupyter notebook文件即可 *.ipynb 


							jupyter notebook

使用!来执行shell命令，比如安装numpy： 


							!pip install numpy
							!pip list | grep pandas

IPython Magic命令：运行计时 - %%time 和 %timeit。 


							import numpy
							%timeit numpy.random.normal(size=100)
